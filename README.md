<h1>CV Generator using Django Framework</h1>

<p>This is a simple CV (Curriculum Vitae) generator built using the Django framework. It allows users to create their CV by filling out a form and then generates a downloadable PDF of the CV.</p>

<h4>Features:</h4>
<p>
  <ul>
    <li>User-friendly interface for creating CVs</li>
    <li>Customizable CV sections such as Personal Information, Education, Work Experience, Skills, etc.</li>
    <li>Downloadable PDF of the generated CV</li>
    <li>Easily extensible for adding more features and sections</li>
  </ul>
</p>
<h4>Installation</h4>
<p>
  <ul>
    1. Clone the repository: 
    <br>
    >> git clone https://github.com/your_username/cv-generator.git
    <br><br>
    2. Navigate into the project directory:
    <br>
    >> cd cv-generator
    <br><br>
    3. Install dependencies: 
    <br>
    >> pip install -r requirements.txt
    <br><br>
    4. Run migrations:
    <br>
    >> python manage.py migrate
    <br><br>
    5. Start the development server:
    <br>
    >> python manage.py runserver
    <br><br>
    6. Open your browser and go to 'http://localhost:8000' to view the application.
  </ul>
</p>
